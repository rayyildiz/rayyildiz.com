You will be creating an llms.txt file for a blog. An llms.txt file is a standardized format that helps AI systems understand and work with a website's content by providing structured information about the site, its purpose, and its content.

Here is the blog information:

<blog_info>
    <url>https://rayyildiz.com</url>
    <author>Ramazan AYYILDIZ</author>
</blog_info>

Your task is to generate a properly formatted llms.txt file based on the provided blog information. Follow these guidelines:

**Structure Requirements:**
- Start with a clear site description and purpose
- Include the main topics and categories covered by the blog
- Specify that the blog is bilingual (English and Turkish)
- List key content types (posts, pages, etc.)
- Include any special features or sections
- Add relevant context about the target audience
- Include posting frequency or content update patterns if mentioned

**Bilingual Content Guidelines:**
- Clearly indicate that content is available in both English and Turkish
- Mention if certain content is language-specific
- Note any translation policies or content localization approaches
- Specify the primary language if there is one

**Format Requirements:**
- Use clear, concise descriptions
- Organize information logically with appropriate headings
- Use standard llms.txt formatting conventions
- Keep descriptions factual and informative
- Avoid marketing language; focus on functional descriptions

**Content to Include:**
- Site name and primary URL (if provided)
- Brief description of the blog's main purpose
- Key topics and subject areas covered
- Content types and organization
- Language information
- Target audience description
- Any unique features or specializations

Write your complete llms.txt file inside <llms_txt> tags. The content should be ready to use as-is for the blog's llms.txt file.
